Разработанные отчеты

В настоящее время разработаны следующие отчеты:
Уведомление о прекращении блокирования
Уведомление о поступлении заявки на кредитование
Уведомление о блокировке денежных средств по кредитной заявке
Подтверждение кредитной заявки на участие в аукционе
Уведомление о поступлении отложенной заявки на участие в аукционе
Деньги заблокированные(или комиссия) на площадке


Система отчетов АСТ-Финанс.МФО предназначена для поиска информации в документообороте МФО с Сбербанк-АСТ.
Система состоит из робота, который скачивает необходимые данные из личного кабинета МФО в Сбербанке-АСТ, из базы данных параметров сообщений и системы отчетов.
Система разработана с рассчетом, что новые типы отчетов добавлять несложно.
Если вам необходим какой-то другой отчет по данным из документооборота, обращайтесь к разработчику.
